# Architecture Decision Records

ADRs record material technical decisions for Yggdrasil Chronicles.

Use filenames in the form `NNNN-short-title.md`. Start from `0001`.

Required sections:

```text
# ADR-NNNN: Title
Status:
Date:
Owners:

## Context
## Decision
## Alternatives
## Consequences
## Migration and Rollback
## Affected Contracts
```

ADRs cannot override `AGENTS.md` or another higher-authority canonical contract.
